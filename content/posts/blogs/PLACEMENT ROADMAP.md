# Programming

- Learn the programming language (Java, Python) in [w3schools](https://www.w3schools.com/) or [Youtube](https://www.youtube.com/@TechWithTim).
- Then Practice problems in [HackerRank Alogrithms](https://www.hackerrank.com/domains/algorithms) , [HackerRank DS](https://www.hackerrank.com/domains/data-structures)(Preferred) or [Leetcode](https://leetcode.com/).
- Easy to Medium Level programming problems are enough to learn.
