Protocol and its Port

Protocol Port
HTTP 80
HTTPS 443
FTP 21
SSH / SFTP 22
Telnet 23
SMTP 25
DNS 53
DHCP 67 / 68
POP3 110
IMAP 143
SMTP (Simple Mail Transfer Protocol) is used for sending emails.
POP3 is used to receive emails by downloading them from the server.
IMAP allows accessing emails directly on the mail server.
Class Range (1st Octet) Network / Host Bits Use
Class A 1 – 126 8 / 24 Very large networks
Class B 128 – 191 16 / 16 Medium networks
Class C 192 – 223 24 / 8 Small networks
Class D 224 – 239 — Multicasting
Class E 240 – 255 — Research

Subnetting is the process of:

Dividing a large network into smaller sub-networks (subnets)
